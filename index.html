Relação dos esportes eletrônicos com o mercado de games

...

Descrição do projeto

O presente projeto tem o objetivo de demonstrar a relação entre as premiações de esportes eletrônicos e a venda de jogos eletrônicos. O projeto é baseado em informações contidas no site de esportes eletrônicos esportsearnings.com [1] e as vendas de videogames lançados antes de 2017, de acordo com o conjunto de dados Videogame Sales em kaggle.com [2] . Através das visualizações será possível verificar se há alguma relação com o valor de premiações e a quantidade de vendas.


Descrição dos dados

Os dados sobre esportes eletrônicos foram retirados da fonte esportsearnings.com [1] , esta base traz informações sobre torneios de eSports atuais e anteriores, com informações sobre o tamanho do prêmio, número de competidores, número de torneios, país que ganha mais e melhores jogadores. Como não há a possibilidade de download diretamente dos dados do site, será feita a extração das informações necessárias usando o Web Scraping [3].

Os dados sobre as vendas sales.csv apresentam um conjunto de dados de vendas de videogame em kaggle.com [2] . Este conjunto de dados categorizou os videogames lançados antes de 2017 e os organizou por desenvolvedor, editor, plataforma, data de lançamento, classificação ESRB, vendas em diferentes regiões e pontuações do Metacritic por críticos e usuários.

Posteriormente será realizado a combinação entre as duas bases de dados, eliminando as dimensões desnecessárias, para o objetivo do projeto, os nomes dos jogos serão comparados entre as duas planilhas e serão mantidas para análise apenas os jogos que existem nas duas bases de dados


Perguntas de partida

Algumas perguntas iniciais foram elaboradas para dirigir o desenvolvimento das visualizações:

As vendas de jogos fazem diferença no total de prémios dos campeonatos?

Quais tipos de jogos possuem maior premiação?

O principal país para campeonatos afeta as vendas de jogos nesta região?


Metodologia

Foi feita a análise inicial dos dados para verificar as dimensões e as relações que serão usadas para os objetivos e perguntas base do projeto. Os dados filtrados foram alocados em um repositório gist.

As visualizações foram construídas usando a linguagem JavaScript (JS) para a criação das visualizações gráficas, incorporando as seguintes bibliotecas: Data Driven Documents (D3), Crossfilter, Dimensional Charting (DC) eVega-embed [4] e Vega Vega-Lite são ferramentas de visualização que implementam uma gramática de gráficos, semelhante ao ggplot2.

Foi utilizado como ambiente para simulações o ObservableHQ que utiliza o paradigma de notebooks e interage muito bem com bibliotecas JS para gerar as visualizações que posteriormente foram indexadas a página principal de visualização, essa metodologia permite a alteração das visualizações de forma independente do código da página web.

Por fim, foi construído um site com base nas tecnologias HTML, CSS, e JavaScript para estruturação da página, estilização e dinamicidade respectivamente. Esse site interativo será disponibilizado no Github Pages.

Implementação

Após a análise inicial dos dados foram selecionadas as dimensões a serem usadas na visualização. Do conjunto de dados de vendas de videogame do Kaggle [2] , estamos usando apenas os nomes dos videogames, as dimensões relacionadas às vendas (vendas globais, vendas NA, vendas na UE, vendas no Japão e outras vendas) e o gênero de jogo para diferenciação nos canais de cores.

A partir dos dados dos eSports [1], estamos usando os nomes dos videogames (que se correlacionam com o conjunto de dados de vendas de videogames), o número de competidores, o prêmio em dinheiro e o país com z.

Depois de obter nossos conjuntos de dados brutos, os arquivos .csv foram armazenados no Gist, que é uma ferramenta que permite o compartilhamento de pequenos trechos de código. Há também quem use o Gist para mostrar textos e opiniões, com essa ferramenta foi possível acessar os dados sem a necessidade de ter os arquivos no host local.

Foi realizada a combinação dos dados de forma que eles se correlacionarem. A primeira etapa foi eliminar todos os jogos de nosso conjunto de dados de eSports que não estavam presentes no conjunto de dados de vendas de videogame (já que a maioria dos dados vinha de lá). Através da localização das entradas no conjunto de dados Videogame Sales e copiando-as para uma nova planilha. A partir daí, foi aplicada a combinação dos jogos de mesmo nome, já que o conjunto de dados Videogame Sales tinha entradas repetidas para diferentes plataformas. Para fazer isso, foi feita a combinação das vendas de cada plataforma, calculando a média das pontuações dos críticos e das pontuações do usuário (já que essas pontuações às vezes eram diferentes para plataformas diferentes) e eliminando quaisquer dimensões desnecessárias (como editor e plataforma).

Também foi feito um tratamento para a data de lançamento mais antiga, como alguns jogos lançados em anos diferentes (um exemplo seria a versão para PC do jogo lançado um ano após as versões para Xbox e Playstation). A partir daí, foi decidido combinar nossos dados em um conjunto de dados, tornando-o mais fácil de usar no notebook Observable.

Foram adicionadas as dimensões do conjunto de dados de eSports às entradas correspondentes. As dimensões transferidas foram o número de jogadores, o número de torneios, o prêmio em dinheiro, o melhor país e a região. Esta foi a inicial do conjunto de dados originalmente. No entanto, houve um problema. Os valores do prêmio em dinheiro incluíam um cifrão, e isso não funcionou no Observable, pois não nos permitiu classificar nosso gráfico de dispersão em ordem crescente por prêmio em dinheiro. Para corrigir isso, a última alteração que fizemos nos dados foi remover o cifrão da dimensão do prêmio, deixando assim como conjunto de dados finais: noDollar .

A primeira visualização apresentada traça a relação entre as vendas globais de cada videogame em nosso conjunto de dados e o prêmio em dinheiro total de eSports concedido para cada jogo. Foi utilizado um gráfico de dispersão para esta visualização porque há uma quantidade relativamente alta de jogos para exibir e após a análise de modelos de gráficos observou-se que normalmente é mais fácil descobrir tendências e relações de gráficos de dispersão do que outros métodos de visualização baseados em texto.

O gênero de cada videogame também é exibido na forma de variações de cores dos pontos. Essa visualização fornece essencialmente aos usuários uma maneira de explorar rapidamente a relação entre as vendas globais de videogames e prêmios em dinheiro em torneios de eSports, enquanto também vê quais gêneros são mais populares e cumulativamente contêm prêmios mais altos de eSports. Passar o mouse sobre um ponto no gráfico de dispersão revela o título do videogame, o valor dos dados de vendas globais e o valor dos dados do prêmio em dinheiro. Além disso, os usuários podem destacar uma seção da visualização por meio da técnica de Brush, e a visualização a seguir fornece uma visão mais ampliada e legível dos dados destacados.

A segunda visualização é uma versão ampliada da visualização anterior, com base nos dados que o usuário destaca ao selecionar a área de destaque com o Brush. Esse recurso de zoom foi bastante útil devido ao alto volume de videogames na primeira visualização e sua aparência desordenada e talvez indistinta, isso pode ser observado nos jogos que têm menor prêmio em dinheiro total em nosso conjunto de dados. Além disso, espera-se que os usuários destaquem um conjunto de dados na visualização ampliada. Eles podem escolher passar por cima de um ou mais videogames, e isso leva ao restante das visualizações de forma dinâmica.

A seleção anterior de jogos em que o usuário destaca com Brush altera e atualiza as 06 visualizações a seguir:

Na Terceira visualização: Number of eSports Tournaments in countries: Hover to view the amount of tournaments for each genre and each game, os usuários veem um gráfico de barras empilhadas que demonstra A quantidade de torneiros realizados por paísTop_Country., além disso é possivel verificar quais são os gêneros preferidos de games Genre. para realização dos torneios em cada país.

Este gráfico de barras pode fornecer informações sobre quantas pessoas o prêmio em dinheiro foi distribuído para o jogo de interesse, bem como qual país reina superior nos torneios do jogo em geral :Top_Country.

Os usuários podem ver e comparar facilmente o número de jogadores entre diferentes jogos, observando as alturas das barras. Nas visualizações 5,6, 7 e 8 North American, European, Japanese and Other Region Sales of Previously Selected Video Games, novamente, com base na seleção dos jogos nas visualizações anteriores, as vendas são reveladas e divididas em regiões. Os usuários podem analisar essas visualizações para determinar como as vendas do jogo são divididas ao redor das regiões e talvez explorar as relações entre os principais países de eSports de videogames e as vendas desses videogames nessas regiões (Em milhões de dólares).

As visualizações 09 e 10 demonstraram a relação das vendas dos jogos selecionados com a avaliação dos usuários e da organização do site Metacritic

Após a criação das visualizações no notebook Observable utilizamos o recurso de incorporação para inserir as visualizações em uma página html e css para visualização final dos usuários, isso permite ter uma maior independência dos arquivos utilizados para visualização


Conclusões

A partir dos resultados obtidos com as visualizações e as questões iniciais da pesquisa podemos ter algumas inferências:
    A premiação não tem relação direta com a as vendas de um jogo (Entretanto não levamos em consideração a quantidade de micro transações ou popularidade)
  • Jogos no estilo shooter e Fighting demonstraram no geral um melhor desempenho em vendas e em Esports (premiação e Quantidade de Campeonatos)
  • desempenho em Esports dos países (Quantidade de campeonatos, domínio, e quantidade de jogadores) influenciam as vendas nas regiões principais)
Como resultado do projeto conclui-se que as visualizações construídas podem ser um bom ponto de partida para explorar os dados de vendas de videogames recuperados do Kaggle e os dados de eSports recuperados do esporte earnings.net. Os usuários podem explorar relacionamentos potenciais entre prêmios de eSports, países que jogam melhor e efeitos nas vendas de videogames.
